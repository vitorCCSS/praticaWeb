Escrevendo um parágrafo em HTML

Está Tag define um parágrafo para um texto escrito em uma pagina
HTML podendo este conter apenas uma ou mais linhas. Perceba também
que utilizamos a Tag "br" antes do parágrafo, para pular linha do trecho
anterior para este e também na produção do texto para pular as linhas.
Há outro detalhe, veja que utilizamos a tag "strong" paraa grifar em negrito.
